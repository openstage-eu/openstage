# openstage

Typed Python models for EU legislative data. Provides Pydantic models for procedures, events, and documents with built-in support for multilingual text and multi-scheme identifiers.

Part of the [openstage project](https://github.com/openstage-eu).

## Installation

Not yet on PyPI. Install from GitHub:

```bash
pip install openstage @ git+https://github.com/openstage-eu/openstage.git
```

## Documentation

Full documentation: [openstage-eu.github.io/openstage](https://openstage-eu.github.io/openstage/)

## Development

```bash
pip install -e ".[dev]"
pytest
```
